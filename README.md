# Sharing Cohorts Foundation Website

A responsive website for Sharing Cohorts Foundation, a non-profit organization dedicated to serving communities in need across Zambia.

## About the Organization

Sharing Cohorts is a non-profit organization founded by inspired youth with a mission to serve the needy and make a positive impact in communities. The organization focuses on outreach efforts including:

- Children's orphanage visits
- Grocery distribution to families in need
- Health and wellness programs
- Medical assistance and health education
- Hospital support services

## Website Features

- **Home Page**: Welcome section with organization's mission and theme poem
- **Our Services**: Photo gallery showcasing outreach efforts in various locations (Lusaka, Kitwe, Ndola)
- **About Us**: Information about the organization and its mission
- **Contact Page**: Ways to get in touch and donate
- **Donation Information**: Contact details for those wishing to support the cause

## Technologies Used

- HTML5
- CSS3 (external stylesheet: `/css/indext.css`)
- Responsive design for mobile and desktop
